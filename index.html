วิธีเชื่อมข้อมูลระหว่างโมดูล

  • แต่ละโมดูลมีปุ่ม "ส่งออก" ที่จะคัดลอกข้อมูลเป็น JSON ไปยังคลิปบอร์ดอัตโนมัติ
  • เปิดโมดูลถัดไป แล้ววาง JSON ลงในช่อง "นำเข้า" ด้านบนของหน้านั้น
  • ข้อมูลทั้งหมดบันทึกอยู่ในเบราว์เซอร์ของเครื่องนี้เท่านั้น (ไม่หายเมื่อปิดหน้าเว็บ แต่ไม่ sync ข้ามเครื่อง)
  • Backend Scraper (โฟลเดอร์แยก) รันบนเครื่อง/เซิร์ฟเวอร์ต่างหาก แล้วส่งข้อมูลเข้า Supabase ให้ Trend Scanner ดึงมาซิงค์ได้
